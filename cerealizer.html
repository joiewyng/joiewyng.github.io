Cerealizer
ROLE : Full Stack Developer
TOOLKIT : Python, HTML, CSS, Heroku
TIMELINE : Spring 2021

OVERVIEW

Cerealizer provides personalized cereal recommendations based on user input keywords such as "chocolate, crunchy", or "marshmallows". Selecting 'similar cereal' returns relevant results on cereal brand input. Results include cereal specific details such as rating and nutritional information. Clicking on a cereal result will allow users to view additional cereal details as well as consumer reviews.
