OVERVIEW

The Currents system is comprised of three main components: a thermal model to determine when to turn on/off the conditioning to a space, a machine learning algorithm used to actively predict a room’s occupancy and push/pull data from a central server, and a smartphone application that inputs a user’s calendar data into the server to better predict for a room’s occupancy as well as provide for manual control of a room’s HVAC system when necessary.
